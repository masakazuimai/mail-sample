# お問い合わせフォーム サンプル

PHPで作成したお問い合わせフォームのサンプルです。

## 仕様

- **PHP** + **PHPMailer**
- 入力 → 確認 → 送信完了の3ステップ
- セッションによるフォームデータの保持
- ファイル添付対応
- GA4イベントトラッキング（`sample_contact_form`）

## ファイル構成

| ファイル | 説明 |
|----------|------|
| index.php | 入力フォーム |
| confirm.php | 確認画面 |
| thanks.php | 送信完了画面・メール送信処理 |
| contact.css | スタイルシート |

## セットアップ

1. `composer install` で依存関係をインストール
2. `thanks.php` のSMTP設定を編集
